## 7.2 快慢指针（删除元素）

```java
// 删除数组中所有等于target的元素，返回新长度
public static int remove(int[] arr, int target) {
    int slow = 0;  // 慢指针：指向"有效数据该放的位置"
    for (int fast = 0; fast < arr.length; fast++) {
        if (arr[fast] != target) {
            arr[slow] = arr[fast];  // 把要保留的元素搬到前面
            slow++;
        }
    }
    return slow;  // slow就是新数组的长度
}

// 为什么用快慢指针？
// 一次遍历完成，O(n)时间，O(1)额外空间，不开新数组
```

## 7.3 二倍均值法（红包算法）

```java
// 公平随机分配红包
public static double[] splitRedPacket(double total, int count) {
    double[] result = new double[count];
    double rest = total;
    Random r = new Random();

    for (int i = 0; i < count - 1; i++) {
        double max = rest / (count - i) * 2;  // 关键：二倍均值
        double money = Math.max(r.nextDouble() * max, 0.01);
        result[i] = money;
        rest -= money;
    }
    result[count - 1] = rest;  // 最后一人拿剩余（防止精度损失）
    return result;
}

// 为什么用二倍均值？
// 保证每个人至少能抢到钱（上限=剩余均值×2）
// 越往后越随机（剩余金额波动大）
// 第一个抢的人不会把所有钱抢光
```

## 7.4 二路归并（找中位数）

```java
// 两个有序数组合并后找中位数
public static double findMedian(int[] arr1, int[] arr2) {
    int[] merged = new int[arr1.length + arr2.length];
    int i = 0, j = 0, k = 0;

    // 选择较小的元素依次放入新数组
    while (i < arr1.length && j < arr2.length) {
        merged[k++] = arr1[i] < arr2[j] ? arr1[i++] : arr2[j++];
    }
    while (i < arr1.length) merged[k++] = arr1[i++];
    while (j < arr2.length) merged[k++] = arr2[j++];

    // 取中位数
    int mid = merged.length / 2;
    return merged.length % 2 == 0
        ? (merged[mid-1] + merged[mid]) / 2.0
        : merged[mid];
}

// 为什么用二路归并而不是排序？
// 两个数组本身有序，归并只需 O(n+m) 时间
// 如果直接合并再排序是 O((n+m)log(n+m))，更慢
```

## 7.5 接雨水（双指针经典题）

```java
public static int trap(int[] height) {
    int left = 0, right = height.length - 1;
    int leftMax = 0, rightMax = 0, water = 0;

    while (left < right) {
        if (height[left] < height[right]) {
            if (height[left] >= leftMax) leftMax = height[left];
            else water += leftMax - height[left];
            left++;
        } else {
            if (height[right] >= rightMax) rightMax = height[right];
            else water += rightMax - height[right];
            right--;
        }
    }
    return water;
}

// 核心思想：一个位置能接多少水，取决于两边最高柱子的较小值减去自身高度
// 为什么双指针？O(n)时间，O(1)空间，一遍扫描完成
// 左右指针向中间逼近，谁小谁先移动（矮的那个决定了当前的水量上限）
```

## 7.6 大乐透（随机不重复）

```java
// 从1~35随机选5个不重复数字
int[] result = new int[5];
Random r = new Random();
int index = 0;

while (index < 5) {
    int num = r.nextInt(35) + 1;
    // 判断是否重复
    boolean exists = false;
    for (int i = 0; i < index; i++) {
        if (result[i] == num) { exists = true; break; }
    }
    if (!exists) result[index++] = num;
}

// 为什么不用ArrayList？
// 数量固定为5，数组足够
// 为什么循环判断而不是用contains？
// 还没学Set集合，用已学知识解决问题
```
