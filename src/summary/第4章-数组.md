# 第4章：数组

## 4.1 为什么需要数组？

```java
// 没有数组：定义100个学生的成绩
int score1 = 85;
int score2 = 92;
// ... 写100个变量？遍历求和？不可能！

// 有数组：
int[] scores = new int[100];
// 100个成绩一个变量搞定，遍历求和轻轻松松
int sum = 0;
for (int i = 0; i < scores.length; i++) {
    sum += scores[i];
}
```

数组解决的核心问题：**用统一的方式管理一组同类数据**。

## 4.2 数组定义与内存

```java
// 静态初始化：已知元素值
int[] arr1 = {10, 20, 30};
int[] arr2 = new int[]{1, 2, 3};

// 动态初始化：已知元素个数，值用默认值
int[] arr3 = new int[5];         // {0,0,0,0,0}
String[] arr4 = new String[3];   // {null,null,null}
```

### 内存图解

```java
int[] arr = {10, 20, 30};

// 栈：arr → 0x100(地址，4字节)
// 堆(0x100)：[10, 20, 30]
//            ↑   ↑   ↑
//          arr[0] arr[1] arr[2]

// arr.length → 3（数组对象自带length属性，在堆中）
```

## 4.3 数组 vs ArrayList（集合）

这是你游戏项目中的实际选择，每个选择都有原因：

| 维度 | 数组 int[] | ArrayList | 
|------|-----------|-----------|
| 长度 | **固定**，创建后不能变 | **动态增长**，自动扩容 |
| 存储类型 | 基本类型+引用类型 | 只能存引用类型 |
| 增删效率 | 低（需要移动元素） | 尾部添加快，中间插入慢 |
| 随机访问 | 快 O(1) | 快 O(1) |
| 内存 | 更省 | 略有开销 |

```java
// 游戏项目中为什么这样选？

// ✅ 用户列表 → ArrayList（用户数量不确定，频繁注册）
ArrayList<User> userList = new ArrayList<>();

// ✅ 验证码字符池 → 固定数组（52个字母+10个数字，固定不变）
char[] pool = new char[62];

// ✅ 敌人列表 → ArrayList（可能会动态增减boss）
ArrayList<Enemy> enemyList = new ArrayList<>();

// 一句话：数量固定的用数组，可能变化的用集合
```

## 4.4 常见问题

### 索引越界

```java
int[] arr = {1, 2, 3};
System.out.println(arr[3]);  // ❌ ArrayIndexOutOfBoundsException
// 3个元素，索引是 0,1,2，没有3
```

### 空指针

```java
int[] arr = null;      // arr没有指向任何数组
// arr[0];             // ❌ NullPointerException
// 操作数组前确保不是null
if (arr != null) {
    System.out.println(arr[0]);
}
```

## 4.5 常用算法

```java
// 1. 求最大值
int max = arr[0];
for (int i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
}

// 2. 反转数组（双指针：一头一尾，向中间走）
for (int i = 0, j = arr.length - 1; i < j; i++, j--) {
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}
// 为什么用双指针？O(n)时间复杂度一次遍历完成，不用额外数组

// 3. 打乱数组（Fisher-Yates洗牌）
Random r = new Random();
for (int i = arr.length - 1; i > 0; i--) {
    int j = r.nextInt(i + 1);
    int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;
}
```
